#### CRUD Response code:

CREATE	| CREATE
- ok: 201 CREATED
- fail: 500 INTERNAL SERVER ERROR

READ	| GET
- ok: 200 OK
- fail: 404 NOT FOUND

UPDATE	| PUT
- ok: 200 OK
- fail: 404 NOT FOUND

DELETE	| DELETE
- ok: 204 NO_CONTENT
- fail: 404 NOT FOUND
